# VOX OPS: The Next-Gen Voice Assistant for Gamers 🎙️🎮

![VOX OPS Logo](docs/logo.svg)

**Control your reality.** VOX OPS is a free, open-source, privacy-first AI voice assistant designed for gamers. Execute complex macros, control your system, and immerse yourself in your favorite games with zero latency.

---

## 🚀 v6.4 - System Override (Latest Update)
The latest version introduces long-awaited command management features!
- **Edit on the Fly**: Double-click any command or use the new `Edit Selected` button to modify macros instantly.
- **Context-Aware Control**: Right-click context menus for rapid `Edit` and `Delete` actions.
- **Auto-Save Intelligence**: Your changes are now automatically saved to your active profile. No more manual saves!

---

## 💎 Key Features
- **100% Offline Speech Engine**: Powered by **VOSK AI**, ensuring your voice data stays on your machine.
- **Zero Latency Performance**: Instant command execution optimized for competitive gaming.
- **Visual Macro Builder**: Create complex multi-step sequences (keyboard, mouse, system, TTS) with a modern GUI.
- **Dynamic Profiles**: Seamlessly switch between gaming, creative, and developer profiles.
- **Community Driven**: Share and download community-built profiles directly within the app.

## 📥 Getting Started
[**Download Latest Release (Windows)**](https://github.com/milak-web/vox-ops/releases/latest)

### 🛠️ Quick Installation
1. Download the `VOX_OPS.exe` from the latest release.
2. Place it in a dedicated folder.
3. Run the application and follow the first-run wizard to select your voice model.
4. Start commanding!

---

## 🌟 Why Choose VOX OPS?
VOX OPS is a modern, open-source alternative to paid assistants like VoiceAttack. It is built on cutting-edge Python libraries (`customtkinter`, `vosk`, `pyautogui`) to provide a sleek, high-performance experience without subscriptions or privacy compromises.
